Dark Mode

Turtle Crossing Game Project

Design by Thomas Organ

Return to Portfolio

Year: 2024
Libraries: Turtle Graphics

• A basic turtle crossing game I designed in Python using Turtle graphics.

• The player controls the turtle at the bottom of the screen while several randomly generated cars move across the screen. The goal is to have the turtle cross the road while avoiding the cars.

• The cars increase in speed with each level score. • Includes save setting on level score.
