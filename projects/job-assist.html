Design by Thomas Organ

Return to Portfolio

Year: 2026
Main Libraries: FastAPI, PostgreSQL (SQLAlchemy), Ollama (DeepSeek-R1:14B), JavaScript, Playwright, BeautifulSoup

• Developed and automated, full-stack data pipeline designed to scrape job roles, analyse target company architectures, and generate tailored cover letters from a provided CV

• Engineered a dual-method data system utilising Adzuna API payloads and Playwright & BeautifulSoup web scraping, governed by a local PostgreSQL database applying SQLAlchemy constraints to automatically prevent duplicate job listings.

• Built a fully offline, prompt-engineered AI inference gateway using FastAPI and a local LLM (DeepSeek-R1), specifically prompted to produce strict JSON output schemas for precise data extraction and professional cover letter generation.

• Delivered a decoupled, lightweight JavaScript dashboard that communicates with the backend REST API, allowing for real-time DOM rendering, status tracking, and database syncronisation without page reloads.
