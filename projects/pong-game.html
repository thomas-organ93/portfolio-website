Pong Game Project

Design by Thomas Organ

Return to Portfolio

Designed: 2024
Libraries: Turtle Graphics

• A basic two-player pong game based off table tennis I designed in Python using Turtle graphics.

• The player controls an in-game paddle either from the left or right where they compete against another player controlling a second paddle on the opposite side.

• Players use paddles to hit a ball back and forth until one loses.

• The ball increases in speed swing of the paddle.
• Includes a scoreboard of the match.
